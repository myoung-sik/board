# Board Project

Spring Boot와 JPA를 활용한 게시판 프로젝트입니다.  
게시글 CRUD 및 페이징 기능을 구현하며 Spring MVC 구조와 데이터 흐름을 학습하기 위해 제작하였습니다.

---

# 📌 개발 환경

- Java 17
- Spring Boot
- JPA
- MariaDB
- Gradle
- IntelliJ IDEA

# ✨ 구현 기능

- 게시글 작성(Create)
- 게시글 목록 조회(Read)
- 게시글 수정(Update)
- 게시글 삭제(Delete)
- 게시글 페이징 처리
